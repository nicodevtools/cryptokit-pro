## 🛠️ Tools Included

| # | Tool | What It Does |
|---|------|-------------|
| 💹 | **Live Prices** | Real-time crypto prices via CoinGecko API. Top 100 by market cap. |
| 📊 | **DCA Simulator** | Backtest dollar-cost averaging strategy. See how regular buys would've performed. |
| 🧮 | **P&L Calculator** | Calculate profit/loss including fees, leverage, and funding. Know your numbers before you trade. |
| 📐 | **Position Sizer** | Risk-based position sizing. Never risk more than you plan to. |
| 📋 | **Portfolio Tracker** | Track your holdings. Data stored in your browser (localStorage) — no server, no leaks. |
| 🔐 | **Wallet Generator** | Generate ETH/Polygon wallet addresses. All computation client-side. |
